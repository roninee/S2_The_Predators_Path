### ① Principles｜原则（Principles）

> **这些是“不需要再论证、只需反复执行”的部分**

**P1. 认知冷酷，行为中性（Cognitively cold, behaviorally neutral）**

- 内部判断可以是冷酷的
- 外部表现必须保持中性
- 不提前暴露立场，不赠送情绪价值

**P2. 预测人不会变，但不外化这个判断**

- 人在压力下的行为由结构决定（incentive structure）
- 正确预测 ≠ 正确表达
- 判断留在模型里，不转化为态度

**P3. 热情默认是负期望值（negative expected value）**

- 在未过滤对象前，热情=额外成本
- 成本包括：时间、精力、情绪回撤
- 只有通过过滤后，热情才可能转为正期望

**P4. 热情是“奖励”，不是“入场券”**

- 热情只对极少数对象释放
- 它是结果，不是前提

### ② Evidence｜证据 / 论证（Evidence / Rationale）

> **这是你说“我必须保留 1/3 的原因”——你需要它来防止自我怀疑**

**E1. 人在压力下必然背叛，是结构性结果**

- 并非道德判断，而是激励约束（incentives & constraints）
- 类似金融危机中的道德风险（moral hazard）

**E2. 中性并非虚伪，而是信息最小化策略**

- 情绪表达=信息泄露（information leakage）
- 中性行为最大化可选性（optionality）

**E3. 热情被系统性误读为“可持续供给”**

- 一旦形成预期，就会被持续索取
- 索取不需要恶意，只需要习惯

**E4. 交易员视角的类比**

- 不因一次盈利就放松风控
- 不因一段关系“感觉不错”就提高敞口

### ③ Failure Zones｜禁区 / 破功点（Failure Zones）

> **这是最有价值的一段，决定你是否反复犯错**

**F1. 把理解误认为“被理解”**

- 你理解别人 ≠ 别人理解你
- 理解本身不是交换品

**F2. 把中性升级为冷漠（overcorrection）**

- 中性 ≠ 冷嘲 ≠ 防御性攻击
- 行为要“可持续”，不是发泄

**F3. 在未过滤前提前投资（premature investment）**

- 提前给建议
- 提前共情
- 提前站队

**F4. 用价值观为越界行为辩护**

- “我一向热情 / 真诚”
- 这是典型的执行失败叙事
