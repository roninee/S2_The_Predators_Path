## 为什么不用 Diplomacy / Game Theory

**Diplomacy（外交）的问题**

- 暗含：
  - 长期互动
  - 高频博弈
  - 社交能力是核心资产
- ❌ 你不是靠“社交优势”获利的人

**Game Theory（博弈论）的问题**

- 过度抽象
- 容易诱发过度推演（overthinking）
- 与你“尽量少消耗认知资源”的目标冲突

------

### Filter 的本质优势（非常关键）

**Filter = 先决策“要不要进入系统”，而不是“如何玩好系统”**

这点和你交易员身份高度**同构**：

- **不是研究所有股票**
- 而是先做 **Universe Selection（交易标的筛选）**

**Filter 的隐含假设：**

- **大多数对象不值得深入**
- 默认策略是“低交互、低暴露”
- 深度是例外，不是常态

这**完美匹配内向 + 交易员**。
