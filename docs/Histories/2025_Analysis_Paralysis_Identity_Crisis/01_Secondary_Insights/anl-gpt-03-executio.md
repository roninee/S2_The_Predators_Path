# execution.md

> **Purpose**：在任何状态下，**减少思考分支，强制进入行动或退出**
> **原则**：*宁可笨，也不允许卡住*

------

## 0️⃣ 总规则（Meta-rule）

**If 我开始“想规则是否合理”超过 30 秒 → Then 立刻执行当前规则，不允许修改。**

> 理由：
> 对你来说，**规则被执行一次的价值 > 被想对一百次**。

------

## 1️⃣ 启动困难（Start Resistance）

### SOP 1.1｜零摩擦启动

**If 我不想开始 / 拖延 / 想“再等等” → Then 只做 2 分钟版本。**

- 不允许升级
- 不允许优化
- 不允许“既然做了就多做点”

> 这是 **Minimum Viable Action（最小可行动作）**

------

### SOP 1.2｜身体先行

**If 脑子卡住 → Then 先动身体 60–120 秒。**

任选其一：

- 走动
- 俯卧撑 / 深蹲
- 放音乐随便动

**Then** 再回到任务。

> 证据基础（英文）：
>
> - Embodied cognition（具身认知）
> - Action precedes clarity（*行动先于清晰*）

------

## 2️⃣ 决策过载（Overthinking）

### SOP 2.1｜三选一封顶

**If 我在 A / B / C 之外继续想 D / E → Then 强制只在前三个里选。**

- 不选“再等等”
- 不选“再收集信息”

> 这是 **Option Capping（选项封顶）**

------

### SOP 2.2｜时间硬截止

**If 决策超过预设时间（如 5 分钟） → Then 选“最不优雅但可执行”的那个。**

> 原则：
> **Ugly but executable > Elegant but delayed**

------

## 3️⃣ 执行中犹豫（Mid-action Doubt）

### SOP 3.1｜不中断原则

**If 执行中开始怀疑 → Then 不允许停，只允许“继续 or 结束”。**

- 禁止：暂停→思考→调整
- 允许：做完 → 复盘

------

### SOP 3.2｜失败不是信号

**If 结果不好 → Then 默认是噪声，不立刻改规则。**

- 至少 **N = 5 次** 才允许调整（N 可自己定，但必须 ≥3）

> 这是 **Noise Tolerance（噪声容忍）**

------

## 4️⃣ 情绪波动（Emotion Spike）

### SOP 4.1｜情绪 ≠ 指令

**If 情绪强烈（兴奋 / 恐惧 / 愤怒） → Then 禁止做任何不可逆决策。**

- 不加仓
- 不删系统
- 不发重要信息

------

### SOP 4.2｜冷却动作

**If 情绪 > 阈值 → Then 执行固定冷却动作（10–20 分钟）：**

- 洗脸 / 洗澡
- 快走
- 音乐 + 闭眼

**Only after 冷却 → 才能回到规则。**

------

## 5️⃣ 复盘过度（Over-reflection）

### SOP 5.1｜只问一个问题

**If 想复盘 → Then 只允许回答一个问题：**

> **“下次我是否还会做同样的动作？”**

- 是 → 保留
- 否 → 改一条规则

------

### SOP 5.2｜禁止情绪复盘

**If 复盘内容开始评价自己（聪明 / 愚蠢 / 自律） → Then 立刻停止。**

> 只允许：
> 行为 → 结果 → 下次是否重复

------

## 6️⃣ 信息摄入（Information Intake）

### SOP 6.1｜输入即责任

**If 我主动看了一条信息 → Then 我必须回答：**

> “这条信息会改变我接下来 24 小时的一个动作吗？”

- 否 → 标记为噪声，下次减少
- 是 → 立刻写成 If–Then

------

## 7️⃣ 连续失败 / 状态差（Bad Streak）

### SOP 7.1｜降维运行

**If 连续 2–3 次状态差 → Then 进入“最低功率模式”。**

- 只保留 1 个核心任务
- 其余全部暂停
- 不做长期规划

------

### SOP 7.2｜不自救原则

**If 想“重塑自己 / 重新理解人生” → Then 一律拒绝。**

> 状态差时：
> **只能靠规则，不能靠顿悟。**

------

## 8️⃣ 当你想“改系统”的时候（Very Important）

### SOP 8.1｜系统冻结

**If 我想大改 execution.md → Then 冻结 7 天。**

- 7 天内只记录想法
- 不实施
- 到期再决定

> 防止 **System Thrashing（系统抖动）**

------

## 9️⃣ 每日最小闭环（Daily Close）

### SOP 9.1｜一天只完成一件事

**If 今天很乱 → Then 只完成 1 件“可完成的小事”，然后收工。**

> 完成感 > 完美感

------

## 结尾硬规则（最后一条）

**If 我没有执行任何规则 → Then 明天不允许新增规则，只允许执行旧的。**
